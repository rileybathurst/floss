# Astro-Style TypeScript CLI Chooser

A tiny command-line app written in TypeScript that shows a styled two-option menu.

## What it does

When you run it, you can choose one of two lines:

- `hello`
- `world`

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Build

```bash
npm run build
```

## Optional global use

After building, you can link the CLI command locally:

```bash
npm link
astro-chooser
```
